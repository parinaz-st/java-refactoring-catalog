# Inline Function / Inline Method

Inline a method when its body is as clear as its name, or when you want to undo a poor extraction and refactor it properly.(Fowler, 2nd Edition, p.115)

**Golden Tip:** If this is a method in a class that has subclasses which override it, then do not inline it.
